# Auto Repair Scripts

## 使用方法
1. 查看可用脚本: ls scripts/auto_repair/
2. 运行脚本: python3 scripts/auto_repair/<script_name>.py

## 脚本列表

| 脚本 | 功能 |
|------|------|
| `repair_api_timeout.py` | 修复 API 超时问题 |
| `repair_cron.py` | 修复 Cron 服务问题 |
| `repair_db_connection.py` | 修复数据库连接问题 |
| `repair_dependencies.py` | 修复依赖缺失问题 |
| `repair_permission.py` | 修复权限问题 |
| `repair_port.py` | 修复端口占用问题 |
| `repair_disk_space.py` | 修复磁盘空间不足 |
| `repair_oom.py` | 修复内存溢出问题 |
